
# demo-js-crud 增删改查demo

使用野狗进行数据管理，只需要基本的DOM操作和简单的API就可以进行数据增加，持久化查询，数据修改和删除。


## 在线示例
http://crud.wilddogapp.com/

[![一个demo页面的快照](crud.png)](http://crud.wilddogapp.com/)

## 本地运行
本地运行需要依赖node.js和gulp。
需要安装node.js和npm。

安装环境：

```
cd demo-js-crud
npm install -g gulp

npm install
bower install
```

启动http服务：

```
gulp serve
```

访问浏览器地址: http://127.0.0.1:9000/ 。
